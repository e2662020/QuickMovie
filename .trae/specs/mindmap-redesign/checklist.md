# 验收清单：故事版思维导图节点功能重做

## 视觉设计检查

- [ ] 深色专业级界面风格，背景色为深灰色 (#1a1a2e 或类似)
- [ ] 节点采用圆角矩形设计，带微妙阴影
- [ ] 节点颜色鲜艳但不刺眼，有发光效果
- [ ] 连接线采用贝塞尔曲线，线条流畅
- [ ] 图标使用 Lucide 或类似现代图标库
- [ ] 字体清晰易读，层次分明

## 交互功能检查

- [ ] 点击"添加节点"按钮能成功创建新节点
- [ ] 节点可以拖拽移动，位置实时更新
- [ ] 双击节点可以编辑标题
- [ ] 点击节点删除按钮弹出确认对话框
- [ ] 确认删除后节点及其连接线被移除
- [ ] 鼠标滚轮可以缩放画布
- [ ] 画布空白区域可以拖拽平移
- [ ] 重置视图按钮可以将画布恢复到初始状态
- [ ] 颜色选择器可以更改节点颜色

## 连接线功能检查

- [ ] 可以从节点连接点拖拽创建新连接
- [ ] 连接线自动使用贝塞尔曲线
- [ ] 点击连接线可以高亮显示
- [ ] 可以删除选中的连接线

## 节点类型检查

- [ ] 支持角色节点（蓝色系）
- [ ] 支持场景节点（绿色系）
- [ ] 支持事件节点（橙色系）
- [ ] 不同类型节点有对应图标

## 性能检查

- [ ] 节点拖拽流畅无卡顿
- [ ] 缩放操作平滑
- [ ] 50+ 节点时仍保持流畅
- [ ] 无明显内存泄漏

## 错误处理检查

- [ ] 删除节点时正确断开所有连接
- [ ] 编辑时按 ESC 取消编辑
- [ ] 网络错误时显示友好的错误提示
